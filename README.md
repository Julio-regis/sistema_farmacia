# 💊 Sistema Farmácia API

API REST desenvolvida em Java com Spring Boot para gerenciamento de medicamentos.

## 🚀 Tecnologias utilizadas

* Java 17
* Spring Boot
* Maven

## 📌 Funcionalidades

* Listar medicamentos
* Cadastrar medicamentos

## 🔗 Endpoints

### Listar medicamentos

GET /medicamentos

### Cadastrar medicamento

POST /medicamentos

Exemplo de JSON:

```json
{
  "id": 1,
  "nome": "Dipirona",
  "preco": 10.0
}
```

## ▶️ Como rodar o projeto

1. Clone o repositório
2. Abra no IntelliJ
3. Execute a classe principal
4. Acesse:
   http://localhost:8080/medicamentos
